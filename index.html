Mes projets

Booki

Ma mission sur ce projet consistait à concrétiser les concepts élaborés par l'UI designer en les intégrant avec HTML et CSS.

L'objectif principal était de créer un site web réactif qui puisse s'adapter à toute les tailles d'écrans.

Ohmyfood

Ce projet a été pensé en privilégiant une approche "mobile first" pour garantir une expérience utilisateur optimale sur tous les appareils.

Des animations ainsi qu'un loading ont été intégrées pour rendre le site plus interactif. Le code à été organisé et optimisé grâce à Sass.

Sophie Bluel

J'ai été chargé de concevoir le portfolio de l'architecte Sophie Bluel. J'ai créé une page de présentation des travaux, avec des options de filtrage par catégorie, ainsi qu'une page de connexion pour l'administrateur

J'ai utilisé JavaScript pour générer dynamiquement les éléments en communiquant avec une API, permettant ainsi d'ajouter ou de supprimer des projets.

Kasa

J'ai travaillé sur la refonte d'un site de location d'appartements en utilisant React.

J'ai créé les composants et les routes nécessaires, et j'ai également intégrée de manière dynamique les hôtes, les descriptions et les évaluations de chaque logement.

ArgentBank

Il m'a été demandé, pour ce projet, d'intégrer le site avec React et de mettre en place la connexion utilisateur.

J'ai donc mis en place la gestion des connexions des utilisateurs, l'affichage de leurs comptes ainsi que la possibilité de modifier leur pseudo. J'ai également créé de nouvelles routes API pour la future gestion des transactions.
